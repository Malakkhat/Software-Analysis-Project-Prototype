# Software-Analysis-Project-Prototype
## Party Planners Event Cost Planner

## Project Overview
This application helps customers estimate the cost of renting event equipment from Party Planners Inc. It provides a centralized way to browse equipment, select needed items, and calculate total rental costs for an event.

## Problem Statement
Customers currently do not have an easy way to organize equipment needs and estimate costs before renting. This system solves that problem by offering a centralized event cost planner.

## Goals
- Centralize event rental equipment information
- Help customers estimate rental costs quickly
- Reduce manual searching and calculation time

## Features
- View available equipment
- Add items to an event list
- Enter quantities
- Calculate estimated total price
- Display a summary of selected rentals

## Users
Potential customers of Party Planners Inc.

## Future Enhancements
- Customer accounts
- Save event plans
- Export estimate to PDF
- Admin inventory management
